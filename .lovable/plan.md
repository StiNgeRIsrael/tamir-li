

# סידור מחדש של דף הבית

## מבנה חדש של הסקשנים (מלמעלה למטה)

1. **Hero + Stats** -- נשאר כמו שהוא (שורה אחת בדסקטופ)
2. **כלים לפי קטגוריה** -- נשאר כמו שהוא
3. **Premium Banner** -- נשאר
4. **"למה בוחרים בתמיר לי"** -- סקשן עצמאי ברוחב מלא, 4 כרטיסים בשורה
5. **פורמטים נתמכים + איך זה עובד** -- זה לצד זה בדסקטופ (grid 2 עמודות)
6. **המלצות משתמשים** -- סקשן עצמאי, 3 כרטיסים בשורה אחת בדסקטופ
7. **שאלות נפוצות** -- סקשן עצמאי, 3 עמודות בדסקטופ (2 שאלות בכל עמודה)
8. **טקסט SEO** -- מועבר לפוטר כעמודה חמישית או כבלוק טקסט מתחת לקישורים

## שינויים בקבצים

### `src/pages/Index.tsx`
- פירוק הסקשן המשולב "Features + Formats" -- Features הופך לסקשן עצמאי ברוחב מלא
- יצירת סקשן חדש "Formats + How it works" זה לצד זה (`xl:grid-cols-2`)
- פירוק הסקשן "How it works + Testimonials" -- Testimonials הופך לסקשן עצמאי עם `grid-cols-3` בדסקטופ
- FAQ הופך לסקשן עצמאי עם `xl:grid-cols-3` (6 שאלות מחולקות ל-3 עמודות)
- הסרת בלוק ה-SEO text מהדף (מועבר לפוטר)

### `src/components/SiteFooter.tsx`
- הוספת בלוק טקסט SEO מעל הגריד הקיים של הקישורים, או מתחתיו כפסקה רחבה
- הטקסט יוצג ברוחב מלא עם סגנון מותאם לפוטר
